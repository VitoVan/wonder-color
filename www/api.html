Wonder Color API

Text to Color Converter

Open your developer console, copy and paste below:

$.post('/wonder-api', {text: "hello kitty"}, function(data){ renderColor(data); });

Hit Enter then wait a second.

Wanna JSONP? This link should work:

/wonder-api?text=hello kitty&callback=somefunc

* In some case, you may find that colors returned by this API is different with colors on the home page, that is because: Colors on this page is generated on the server with a headless webkit, though colors on the home page is caculated by your browser, they may have different rendering result.
